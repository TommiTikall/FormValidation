Form validation

Generic HTML, CSS and Javascript approach

This form validation relies on the markup.
This means that the javascript will look at the field and its attributes to determine what to validate. Kinda modular.. yes?
Anyways, this is a practical approach for when developing with CMS, or where you don't want to set up a init with options in javascript for each form on page.
